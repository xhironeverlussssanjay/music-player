GIT PUSH SCRIPTS
================

Double-click salah satu file .bat untuk push ke GitHub:

1. auto-push.bat    -> Otomatis dengan timestamp (RECOMMENDED)
2. quick-push.bat   -> Cepat dengan pesan "quick update"
3. push.bat         -> Custom pesan commit sendiri

Semua file .env dan data lokal sudah aman, tidak akan ter-upload.
